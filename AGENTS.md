# Brand Guidelines

When asked to "Apply Brand Color", adhere strictly to the following visual tokens:

- **Primary Page & Component Background Color**: `#F3EEF4`
- **Main Frame Card Background**: `linear-gradient(135deg, #0F0B14, #1A0F1F)`
- **Card Inner Frame Border / Shadow Accent**: `inset 0 0 0 1px rgba(255,255,255,0.04)`
